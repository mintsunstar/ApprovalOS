#### A 상세 — Acceptance 스모크 (브라우저에서)

시크릿/다른 브라우저로 **두 번째 계정**을 쓰면 초대·알림 확인이 쉽습니다.

| ID | 시나리오 | 기대 |
|----|----------|------|
| A | 승인 시작 | 1단계 **승인자** 계정 벨에 `승인 요청` |
| B | 댓글·핀·AI 분석 | **다른** admin에게 알림. Account에서 해당 토글 OFF 시 미발송 |
| C | 프로젝트 마감 / 최종 승인 | 참여자에게 `결과 공개` |
| D | WS 설정 → 초대 링크 복사 → 로그아웃 → `/invite/...` → 가입 | 해당 WS 소속 |
| E | ProjectNew에 이메일 초대 → 링크 복사 → **다른 이메일** 로그인 | 거부 메시지 |
| F | 동일 이메일로 초대 수락 | 프로젝트 멤버·대시보드 진입 |
| G | `/signup?invite=...` | 가입 직후 WS(·프로젝트) 반영 |
| H | WS 설정: 대기 초대 취소 · 멤버 삭제 | 목록 갱신, 삭제 유저 `project_members` 없음 |

기본 스모크(기존): 개발자 입장 → 시안 2장+ → 투표/댓글/승인 흐름도 한 번 더 확인.
