# ApprovalOS — 운영 관리자 모드 보완 계획 (구현 SSOT)

- 작성일: 2026-07-21
- 버전: **v1.0**
- 원안: `approvalos_admin_screen_v1.0`, `approvalos_admin_usecase_v1.0`
- 런타임: **로컬 데모** (`localStorage`). TOTP·실메일·Stripe·실 API 미터링은 **이번 라운드 제외**

---

## 1. 목표

플랫폼 운영용 **슈퍼관리자 콘솔**을 일반 앱(`users.role = admin` WS 관리자)과 **분리**해 데모에서 동작시킨다.

**성공 기준 (Acceptance)**

| # | 시나리오 | 기대 |
|---|----------|------|
| A | `/admin/login` — 운영자 계정 | 대시보드 진입 |
| B | 일반 유저 `/admin` | 로그인으로 리다이렉트 (운영자 아님) |
| C | WS 목록·상세 | 전체 WS 조회, 플랜 변경·정지 |
| D | 사용자 목록·상세 | 정지/해제, 강제 탈퇴(데모), 로그 기록 |
| E | 플랜 한도 수정 | Free/Pro/Ent 수치 저장, 앱 한도 표시와 연동 |
| F | 운영 로그 | 위 액션이 `admin_logs`에 남음 |
| G | 공지·점검 | CRUD + 점검 모드 플래그 (앱 노출은 최소 배너) |

## 2. 원안 대비 수정 (반드시 준수)

| # | 원안 | v1.0 수정 |
|---|------|-----------|
| 1 | `super_admin` vs WS `admin` 혼동 가능 | **`admin_users` 테이블 전용**. `users.role`의 `admin`과 절대 혼용하지 않음 |
| 2 | TOTP 필수 | 데모: UI만 또는 **스킵 가능**. 실 TOTP는 Auth 전환 후 |
| 3 | Free 프로젝트 3개 | **현행 앱 유지: Free 프로젝트 5** (멤버 5). Pro 프로젝트 50·멤버 30. Ent 무제한 |
| 4 | UC-A09 사용자/에러 로그 | **이번 라운드: 운영자 액션 로그만** (`admin_logs`) |
| 5 | 비밀번호 초기화 메일 | **토스트 stub** |
| 6 | 쿠폰·구독 만료 | **Out of scope** (플랜 한도 편집·WS 적용만) |
| 7 | 실 스토리지/Claude 메트릭 | **집계 stub** (프로젝트 수·멤버 수 기반) |
| 8 | 공지 앱 노출 미설계 | `AppLayout`에 **배너 1줄**만 (발행 중 banner 타입) |

---

## 3. 범위

### In Scope
- Phase A: 스키마·localDb·시드 운영자
- Phase B: `/admin/*` 셸·로그인·가드
- Phase C: 대시보드·WS·사용자·플랜·로그
- Phase D: 공지 CRUD·점검 모드·앱 배너
- Phase E: 문서·빌드

### Out of Scope
- 실 TOTP / Auth 비밀번호
- Stripe·쿠폰·플랜 만료일
- 실 API/스토리지 모니터링
- 사용자 행위·에러 로그 테이블
- Impersonation

---

## 4. 데이터 모델 (로컬)

```
admin_users: id, email, name, password_hint(demo), failed_attempts, locked_until, created_at
account_status: 'active' | 'suspended' | 'deleted'  → users.status
workspace_status: 'active' | 'suspended' | 'deleted' → workspaces.status
plan_limits: { free|pro|enterprise: { max_members, max_projects, storage_gb } }
notices, incidents, admin_logs, system_flags: { maintenance: bool, message, until }
admin_session_id (localStorage)
```

시드 운영자: `ops@approvalos.local` / `operator` (화면에 표시)
